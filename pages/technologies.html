Tecnologias

Java

Básico

Conhecimento básico na linguagem de programação Java envolvendo compreensão com a sintaxe, tipos de dados, programação orientada a objetos, métodos, estruturas de dados simples, tratamento de exceções, coleções, entrada/saída, threads básicas e o ambiente de desenvolvimento com eclipse e IntelliJ.

Angular

Básico

Conhecimentos com o framework Angular compreendendo de conceitos como componentes, templates, data binding, serviços, roteamento, HTTP client, observables, forms e ferramentas de desenvolvimento. Estes são meus conhecimentos fundamentais para criar aplicações web dinâmicas e robustas com Angular.

MySql

Básico

Compreensão de bancos de dados relacionais, SQL, modelagem e consulta de dados, manipulação de registros, chaves e índices, funções agregadas, backup e segurança, além de saber configurar e usar o ambiente de desenvolvimento MySQL. Estas são minhas habilidades para gerenciar eficaz com dados em projetos MySQL.

Linux

Intermediário

Administração de sistemas Linux, incluindo navegação e manipulação de arquivos, gerenciamento de pacotes de software, controle de permissões de arquivos e diretórios, configuração de rede, administração de usuários e grupos, agendamento de tarefas, gerenciamento de processos, criação de scripts shell.

Javascript

Intermediário

Compreensão da linguagem, como declaração de variáveis e tipos de dados, uso de estruturas de controle (if, else, switch), criação de funções, manipulação do DOM, lidar com eventos, trabalhar com arrays e loops, noções de assincronia (callbacks) e habilidades básicas de depuração usando o console do navegador.

Docker

Básico

Conhecimento acadêmicos e com cursos na prática em Docker envolvendo a compreensão de containerização, imagens e containers, Dockerfile, Docker Hub, comandos básicos, volumes, redes, Docker Compose, orquestradores e segurança. Esses conceitos me permitem efetivamente criar, distribuir e gerenciar aplicativos em contêineres com Docker.

Typescript

Básico

Envolvimento e entendimento com tipos estáticos, interfaces, classes e herança, módulos e importações, generics, decorators e configuração do TypeScript. Familiaridade com ferramentas de desenvolvimento e integração com o ecossistema JavaScript para desenvolvimento de aplicativos mais robustos e escaláveis.

Figma

Intermediário

Habilidade com criações de interface e de design, organização, estilos, texto e cores para consistência, componentização para reutilização eficiente, prototipagem, colabs e comentários, exportação de recursos, integrações e plugins, controle de versões e histórico, boas práticas de design.

Git

Intermediário

Envolvimento intermediário com branching e merging, resolução de conflitos, rebase interativo e uso de git hooks. Além disso, tenho compreensão de workflows, submódulos, git bisect e revisão de código, permitindo uma colaboração mais eficaz e um gerenciamento sofisticado de versões e releases.

VS Code

Intermediário

Desenvolvimento diário com editor VSCode de código-fonte com o SO Linux. Essa IDE inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código, tornando o desenvolvimento do profissional mais ágil e eficaz.
